==================================================================
Getting and Cleaning Data
Project Course week 3
==================================================================

The course project consist in creating a dataset following tidy data rules from given data coming from an experiment around Human Activity Recognition Using Smartphones Dataset made by smartlabs (www.smartlab.ws)
More details can be found in the README file of the data provided in subfolder "UCI HAR DATASET"

The folder provided includes :
- README.txt
- UCI HAR DATASET Folder : contains the original dataset provided by smarlabs.ws (in subfolder "UCI HAR DATASET")
- measurements_mean : the final tidy dataset called 
- the R program to create the tidy dataset
- a codebook to explain the transformation from original dataset to requested tidy dataset
